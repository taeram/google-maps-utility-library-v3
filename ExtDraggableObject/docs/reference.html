ExtDraggableObject

This sets up a given DOM element to be draggable around the page.

For a description and examples of how to use this library, check out the how-to.

class ExtDraggableObject

Constructor

Constructor Description
ExtDraggableObject(src:HTMLElement, opts?:opt_drag) Sets up a DOM element to be draggable. The options available within opt_drag are: top, left, container, draggingCursor, draggableCursor, intervalX, intervalY, toleranceX, toleranceY, restrictX, and restrictY.
