ExtDraggableObject

This sets up a given DOM element to be draggable around the page.

For a description and examples of how to use this library, check out the how-to.

class ExtDraggableObject

Constructor

Constructor Description
ExtDraggableObject(src:HTMLElement, opts?:ExtDraggableObjectOptions) Sets up a DOM element to be draggable. The options available within ExtDraggableObjectOptions are: top, left, container, draggingCursor, draggableCursor, intervalX, intervalY, toleranceX, toleranceY, restrictX, and restrictY.

class ExtDraggableObjectOptions

This class represents the optional parameter passed into constructor of ExtDraggableObject. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.
