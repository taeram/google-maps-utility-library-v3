Scroll Wheel Zoom Examples

API Reference

Include Scripts

This library enables scroll wheel zoom before it is in core API as a temp solution. It works with Google Maps API V3.

The first step is to include scrollwheelzoom.js or scrollwheelzoom_packed.js in your document header, after GMaps API is load. You can use the online version if you do not want to download the script.

      <script src="/path/to/scrollwheelzoom_packed.js" type="text/javascript"></script>

Enable Scroll Wheel Zoom

You can simply call google.maps.Map.enableScrollZoom() to enable it.

        function init(){
         var myOptions = { 
            zoom: 13, 
            center: new google.maps.LatLng(49.2903, -123.1294), 
            mapTypeId: google.maps.MapTypeId.ROADMAP 
           };
           map = new google.maps.Map(document.getElementById("map"), myOptions); 
          //...
          map.enableScrollWheelZoom();  
        }
