Scroll Wheel Zoom for V3

This Lib allows Scroll Wheel Zoom before it is officially availabel in core API. Will only test in limited browsers.

For a description and examples of how to use this library, check out the how-to.

class Map

These are new methods added to the Google Maps API's Map class.

Methods

Methods Return Value Description
disableScrollWheelZoom() None Disable wheel zoom.
enableScrollWheelZoom() None Enable wheel zoom.
scrollWheelZoomEnabled() Boolean Returns true if the Wheel zoom feature has been enabled.
